VR Remote Control

Use your mobile phone or a daydream controller as a WebVR remote controller.

Remote Page Demo Page

If using a mobile phone as a remote, open the Demo Page on desktop and the Remote Page on your mobile. Pair the devices together using the pair code on the demo page.

Best used with A-Frame. See source code and other components on aframe-polats-extras.  VR Remote Control is based on Don McCurdy's ProxyControls.js.
